Tools & Installation

The tools we need to be effective developers.
Learning Outcomes

On successful completion of this Unit students will be able to,

  1. Install Git, Ruby, Slack, code editor (Atom, VS code)
  2. Install Homebrew, RVM (Mac)
  3. Install Bash (Windows)

Slack

An excellent communication tool for teams!

Download Slack: https://slack.com/

Atom

A text editor packed with awesome features to help us write code.

Download Atom: https://atom.io/

Git

Git is the most commonly used version control system today. More on this next week!

Follow this guide to install and configure Git: https://www.atlassian.com/git/tutorials/install-git

Ruby

A programming language with a focus on simplicity and productivity.

For Mac/UNIX: RVM

For Windows: RubyInstaller

Bash

We'll use commands in bash to create files, run code and loads more!

For Windows: Follow this guide

Homebrew
